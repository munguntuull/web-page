Internet

The Internet is a global network that connects millions of computers and devices together. It allows people to share information and communicate across the world using different services like email and websites.

Web

The Web, or World Wide Web, is a system of websites and web pages that run on the Internet. It uses browsers like Chrome or Safari to access information through links.

How are the Internet and Web different?

The Internet is the physical network that connects computers worldwide, while the Web is a service that uses that network to display websites. In simple terms, the Web is a part of the Internet.

ASCII and Unicode

ASCII is a character encoding system that represents text using numbers, mainly for English characters. Unicode is a larger system that supports many languages and special characters from around the world.

What is UTF-8

UTF-8 is a character encoding format used to store and display text. It is part of Unicode and can represent almost all characters from different languages.

URL

A URL is the address of a webpage on the Internet. It tells the browser where to find a specific website or resource.

Domain Name

A domain name is the readable name of a website, like google.com. It is easier to remember than an IP address.

HTTP

HTTP stands for HyperText Transfer Protocol. It is the system used to transfer web pages from a server to a browser.

HTTPS

HTTPS is a secure version of HTTP. It encrypts the data being sent between the browser and server to keep information safe.

IP Address

An IP address is a unique number assigned to each device connected to a network. It helps devices find and communicate with each other.

DNS

DNS stands for Domain Name System. It translates domain names into IP addresses so browsers can locate websites.

W3C Recommendations

W3C Recommendations are official web standards created by the World Wide Web Consortium. They help ensure websites work properly across different browsers.

XML, XHTML, HTML, HTML5

HTML is the standard language used to create web pages. XHTML is a stricter version of HTML. XML is a language used to store and transport data. HTML5 is the newest version of HTML with improved features for modern websites.
